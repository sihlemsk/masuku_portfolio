My Projects

Fake Job Postings Detection

This project aimed to develop a machine learning model to detect fake job postings by analyzing job details like titles, locations, and company profiles. Using the fake_job_postings.csv dataset, preprocessing handled data imbalance and missing values. The XGBoost Classifier achieved 98.5% accuracy and a 0.79 F1 score post-tuning, showing strong precision and balanced recall crucial for identifying fraudulent postings accurately. Future steps include refining model robustness and deploying it for real-time fraud detection.

Loan Approval Prediction

A loan prediction modeling project aimed at predicting loan approval status based on applicant data. The analysis utilized a dataset containing 381 loan applications with features including demographic, financial, and credit history information. The project evaluated Random Forest and XGBoost classifiers, finding that Random Forest achieved superior performance with 82.26% test accuracy, compared to 79.03% for XGBoost. The modeling pipeline included data cleaning, exploratory analysis, feature preprocessing, and model training. Recommendations for future work include exploring additional algorithms, tuning hyperparameters, and further feature engineering to enhance predictive accuracy.
